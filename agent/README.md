# Agent

Amazon Bedrock AgentCore agent powered by Strands framework and Amazon Nova 2 Lite.

## Setup

```bash
cd agent
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## Development

```bash
agentcore dev
```

## Deployment

```bash
agentcore deploy
```

## Testing

```bash
pytest test/
```
